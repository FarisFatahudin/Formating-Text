APA ITU HTML ?

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web. Penanda atau markup ini, nanti akan kita sebut dengan Tag. Penanda atau markup ini, nanti kita sebut dengan HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web. Kalau kita ibaratkan nih.. HTML itu seperti batu bata untuk membangun rumah. Batu bata ini dapat disusun, hingga menjadi pondasi dasar. HTML memiliki kelebihan seperti memiliki sumber komunitas yang cukup besar dengan pengguna yang luas. Tidak hanya itu, Kelebihan lainya adalah HTML mudah untuk dipelajari oleh seorang Web Developer pemula. Kelebihan lainya adalah HTML merupakan bahasa pemrograman yang gratis atau open source
